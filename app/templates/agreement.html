{% trans "Орган по сертификации ООО «AVTO STANDART ETALON» (далее именуемый – «Орган по сертификации»), с одной стороны, и Организация далее именуемый – «ЗАКАЗЧИК», заключают настоящее соглашение на сертификацию, с другой стороны, именуемые в дальнейшем вместе и/или по отдельности «СТОРОНЫ», заключили настоящее Соглашение на проведение работ по сертификации (в дальнейшем «СОГЛАШЕНИЕ») о нижеследующем:" %}

{% trans '1. Общие положения' %}

{% trans "Настоящее соглашение устанавливает условия сертификации в соответствии требованию п.4.1.2.2. O'z DSt ISO/IEC 17065:2015, с целью проведения работ по подтверждению соответствия продукции." %}

{% trans '2. Предмет соглашения' %}

{% trans "«ЗАКАЗЧИК» поручает, а «Орган по сертификации» принимает на себя обязательства по выполнению работ по обязательной сертификации продукции, указанной в Заявке." %}

{% trans '3. Обязанности сторон' %}

{% trans "3.1 «Орган по сертификации» обязуется:" %}

3.1.1

{% trans 'Оказать услуги по проведению сертификации продукции (далее – услуги) в соответствии с требованиями и сроками, установленными законодательными актами Республики Узбекистан в области технического регулирования, как лично, так и с привлечением третьих лиц, в рамках области аккредитации Органа по сертификации и объективно оценивать полученные результаты.' %}

3.1.2

{% trans 'Требовать от Заказчика предоставления необходимых документов, в соответствии с требованиями установленными законодательными актами Республики Узбекистан, в области технического регулирования и дополнительной информации, о необходимости которых стало известно в ходе исполнения настоящего Соглашения.' %}

3.1.2.2

{% trans 'Орган по сертификации приступает к оказанию услуг после поступления денежных средств от Заказчика в течении 2-х рабочих дней.' %}

3.1.3

{% trans 'Обязуется обеспечить конфиденциальность всей информации об организации или/и продукции ЗАКАЗЧИКА, полученной или созданной в ходе осуществления деятельности по сертификации и использовать ее только для достижения целей, определяемых условиями Соглашения. Не использовать ее полностью или частично в иных, не предусмотренных настоящими условиями Соглашения, целях, без получения предварительного письменного Согласия передавшей Стороны. Уведомить Заказчика о предоставлении конфиденциальной информации, согласно законодательству Республики Узбекистан.' %}

3.1.4

{% trans 'Осуществлять сертификацию продукции не дискриминационно, обеспечивая беспристрастность, объективность, компетентность при принятии решений о выдаче сертификата соответствия продукции.' %}

3.1.5

{% trans 'Приостановить работы, если в процессе их выполнения выявляется неизбежность получения отрицательного результата и нецелесообразность их продолжения, и уведомить об этом Заявителя непосредственно или с использованием средств связи, обеспечивающих фиксированную доставку.' %}

3.1.6

{% trans 'Информировать владельца сертификата об изменениях в процедуре сертификации и схемах сертификации, влияющих на него непосредственно.' %}

3.1.7

{% trans 'Вести учет оформленных, утвержденных, приостановленных или прекращенных сертификатов, с указанием области применения, и ежемесячно предоставлять информацию о них в Агентство.' %}

3.1.8

{% trans 'Предоставлять информацию по статусу сертификата соответствия ЗАКАЗЧИКА по запросу любого заинтересованного лица.' %}

3.1.9

{% trans 'Не передавать оригиналы или копии документов, полученные от Заявителя, третьим лицам без его письменного согласия.' %}

3.1.10

{% trans 'Принимать письменные претензии ЗАКАЗЧИКА, касающиеся процедуры сертификации.' %}

3.1.11

{% trans 'Реагировать на жалобы, поступающие от Заказчика, в установленном порядке.' %}

3.1.12

{% trans 'Предоставить ЗАКАЗЧИКУ результаты выполненных работ.' %}

3.1.13

{% trans 'Устанавливать стоимость работ по Сертификации на основе расчёта трудоёмкости количества человек/дней согласно утверждённому тарифу по ОС.' %}

{% trans '3.2 Заказчик обязуется:' %}

3.2.1

{% trans 'Выполнять установленные законодательством требования Республики Узбекистан в области технического регулирования при сертификации продукции, включая требования к продукции, и реализацию соответствующих изменений, доведённых до него органом по сертификации;' %}

3.2.1.2

{% trans 'Своевременно и в полном объеме осуществлять оплату за оказываемые Органом по сертификации услуги в порядке и на условиях, определенных настоящим договором;' %}

3.2.2

{% trans 'Обеспечить «Орган по сертификации» необходимой достоверной информацией и документами, предусмотренными правилами и/или схемой сертификации, а также законодательством Республики Узбекистан и в соответствии с требованиями Технических регламентов Евразийского экономического Союза (Таможенного Союза) для выполнения обязательств по настоящему Соглашению, посредством сайта «singlewindow.uz».' %}

3.2.3

{% trans 'Обеспечивать выполнение всех обязательных процедур при проведении сертификации продукции (предоставлять доступ представителей органа по сертификации для анализа его состояния (для схемы №3 и №4), отбора образцов продукции для испытаний и т.п., а также доступ к соответствующему оборудованию, местоположению, территории, персоналу и к субподрядчикам заказчика).' %}

3.2.4

{% trans 'Предоставить Органу по сертификации возможность доступа на производство для ознакомления с технологией, процессом изготовления, условиями хранения сырья и готовой продукции (если предусмотрено схемой сертификации).' %}

3.2.5

{% trans 'Обеспечивать стабильность показателей (характеристик) продукции, которые подтверждены при сертификации соответствия, требованиям нормативных документов, а также выполнять установленные требования к объектам подтверждения соответствия, прошедшим процедуру сертификации.' %}

3.2.6

{% trans 'Извещать Орган по сертификации обо всех изменениях, вносимых в техническую документацию или технологические процессы производства продукции, влияющие на ее безопасность, соответствие которой подтверждено.' %}

•
{% trans 'изменения в организации и руководстве (например, изменение в высшем руководстве, персонале, принимающем решения, или техническом персонале);' %}
•
{% trans 'правовой, коммерческий, организационный статус или право собственности;' %}

{% trans 'адрес и места осуществления деятельности;' %}

{% trans 'Вести учёт претензий и жалоб на продукцию, прошедшую процедуру сертификации и извещать о них письменно Исполнителя, а также:' %}

{% trans 'принимает соответствующие меры по отношению к таким жалобам, претензиям и любым недостаткам, обнаруженным в продукции, которые оказывают влияние на соблюдение требований сертификации;' %}

{% trans 'За заявления, касающиеся сертификации продукции, которые могут рассматриваться как непозволительные и вводящие в заблуждения.' %}

{% trans 'За ненадлежащее выполнение сертификационных требований к продукции, нарушение порядка реализации продукции, нарушение порядка маркировки продукции и использования знаков обращения на рынке.' %}

{% trans 'За поставку (реализацию) сертифицированной и маркированной знаком обращения на рынке продукции, не отвечающей требованиям, установленным законодательством Республики Узбекистан и в соответствии с требованиями Технических регламентов Евразийского экономического Союза (Таможенного Союза) или иными нормативными документами, указанными в сертификате соответствия.' %}

6.2.3

{% trans 'За управление всей информацией, полученной или созданной в процессе выполнения работ по настоящему соглашению.' %}

{% trans '6.3. За неисполнение или ненадлежащее исполнение своих обязательств по настоящему Соглашению стороны несут ответственность в соответствии с действующим законодательством Республики Узбекистан.' %}
